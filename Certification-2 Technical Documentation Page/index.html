Introduction

JavaScript is a high-level programming language used to create interactive websites.

It runs in the browser and can modify HTML and CSS dynamically.

console.log("Hello, world!");
  • Dynamic
  • Interpreted
  • Client-side
Setup

To use JavaScript, you can include it in an HTML file using the <script> tag.

You can also load it externally from a .js file.

<script src="app.js"></script>
  • Inline script
  • External file
Syntax

JavaScript uses statements and expressions similar to other programming languages.

Each statement ends with a semicolon.

let x = 5; const name = "John";
  • Variables
  • Functions
